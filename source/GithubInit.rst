**Github初始**

1. GitHub新建Repository

2. 本地克隆下载Github资源库

   ::

       git clone https://github.com/GuokaiLiu/RTD.git

3. 复制粘贴本地资源到资源库文件夹

   ::

       cd RTD
       Ctrl-C + Ctrl-V

4. 添加修改文件

   ::

       git add .

5. 提交修改信息

   ::

       git commit -m 'remark'

6. 上传本地资料

   ::

       git push -u origin master
